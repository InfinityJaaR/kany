CONSULTAS SQL PARA CREAR TABLAS SENCILLAS DE KANY EN SUPABASE
===============================================================

COMO EJECUTAR ESTOS SQL EN SUPABASE
-----------------------------------
1. Entra a https://supabase.com
2. Abre tu proyecto.
3. En el menú izquierdo entra a SQL Editor.
4. Haz clic en New query.
5. Copia y pega las consultas de este archivo.
6. Haz clic en Run.
7. Luego puedes revisar las tablas en Table Editor.

Nota:
- Estas tablas son sencillas y pensadas para empezar rápido.
- El campo id se genera automáticamente.
- El campo created_at se llena automáticamente con la fecha de creación.


1. TABLA: MASCOTAS PERDIDAS
---------------------------

create table lost_pets (
  id bigint generated by default as identity primary key,
  name text not null,
  breed text,
  color text,
  location text,
  date_text text,
  description text,
  reward text,
  contact text,
  status text default 'normal',
  created_at timestamp with time zone default now()
);


2. TABLA: MASCOTAS ENCONTRADAS
------------------------------

create table found_pets (
  id bigint generated by default as identity primary key,
  type text not null,
  breed text,
  color text,
  location text,
  date_text text,
  description text,
  condition text,
  contact text,
  match text,
  status text default 'normal',
  created_at timestamp with time zone default now()
);


3. TABLA: DONACIONES / CAMPAÑAS
-------------------------------

create table campaigns (
  id bigint generated by default as identity primary key,
  title text not null,
  description text,
  goal numeric default 0,
  current numeric default 0,
  donors integer default 0,
  days_left integer,
  organization text,
  status text default 'normal',
  updates integer default 0,
  created_at timestamp with time zone default now()
);


4. TABLA: PRECIOS
-----------------

create table food_prices (
  id bigint generated by default as identity primary key,
  brand text not null,
  weight text,
  price numeric default 0,
  store text,
  available text,
  created_at timestamp with time zone default now()
);


5. TABLA: VETERINARIAS
----------------------

create table vets (
  id bigint generated by default as identity primary key,
  name text not null,
  services text,
  location text,
  phone text,
  hours text,
  promo text,
  created_at timestamp with time zone default now()
);


ORDEN RECOMENDADO
-----------------
1. Ejecuta todas las consultas en Supabase SQL Editor.
2. Verifica que aparezcan estas tablas en Table Editor:
   - lost_pets
   - found_pets
   - campaigns
   - food_prices
   - vets
3. Luego puedes insertar datos manualmente desde Table Editor o conectar el proyecto Next.js con Supabase.
